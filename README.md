# 🌾 Smart Farm Worker Health Risk Prediction using Machine Learning

## 📌 Project Overview
This project presents a machine learning based system designed to predict potential health risks among farm workers. Agricultural workers often face harsh environmental conditions such as high temperature, humidity, and long working hours. These factors can negatively affect their health and productivity.

The developed system analyzes environmental and physiological parameters and predicts the level of health risk using machine learning algorithms. An interactive dashboard is built using Streamlit to visualize predictions and system insights.

---

## 🎯 Problem Statement
Farm workers are exposed to extreme environmental conditions which can lead to serious health risks. Early prediction and monitoring of these risks can help improve worker safety and productivity.

This project aims to develop an intelligent system capable of predicting health risk levels using machine learning techniques.

## 🧠 Machine Learning Models Used

The following algorithms were trained and evaluated:

- Logistic Regression
- Decision Tree
- Naive Bayes
- Support Vector Machine

---

## 📊 Model Evaluation

The model performance was evaluated using multiple metrics:

- Accuracy
- Confusion Matrix
- ROC Curve
- Feature Importance

The results demonstrate strong classification performance and effective risk prediction.

---

## 📂 Dataset Information

The dataset contains farm worker health and environmental parameters including:

- Age
- Temperature
- Humidity
- Heart Rate
- Working Hours
- Health Risk Index

These parameters were used to train machine learning models to classify risk levels.
